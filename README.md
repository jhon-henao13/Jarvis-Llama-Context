# ⚡ JARVIS - Personal AI Assistant (Mark V)

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![Groq](https://img.shields.io/badge/Powered%20by-Groq%20Llama%203.1-orange.svg)](https://groq.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**JARVIS** es un asistente de voz avanzado inspirado en Iron Man, diseñado para automatizar tareas en Windows, controlar música y gestionar flujos de trabajo de programación mediante lenguaje natural y memoria contextual.

---

## ✨ Características Principales

* 🧠 **Memoria Contextual:** Recuerda de qué estabas hablando (Powered by Llama 3.1).
* 🎵 **Stark Music Autoplay:** Automatización de YouTube Music con `pyautogui`.
* 💻 **Modo Dev:** Abre tu entorno de desarrollo (Sublime Text, Gemini, Proyectos) con un comando.
* ⚡ **Latencia Ultra-Baja:** Respuestas casi instantáneas gracias a la infraestructura de Groq.
* 🗣️ **Voz Persistente:** Motor de voz optimizado para evitar bloqueos en Windows.

---

## 🛠️ Instalación

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/TU_USUARIO/Jarvis.git](https://github.com/TU_USUARIO/Jarvis.git)
    cd Jarvis
    ```

2.  **Instala las dependencias:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Configura tus credenciales:**
    Crea un archivo `.env` en la raíz del proyecto y añade tu API Key:
    ```env
    API=tu_groq_api_key_aqui
    ```

4.  **Ejecuta a Jarvis:**
    ```bash
    python main.py
    ```
